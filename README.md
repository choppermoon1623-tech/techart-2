# techart-2

技術・美術（techart）の授業向け教材を、1ファイルにまとめた単一ページ（バンドル済みHTML）です。画像などのアセットをファイル内に埋め込んでいるため、サーバーやインターネット接続なしで、ブラウザ単体で動作します。

🔗 **公開ページ:** https://choppermoon1623-tech.github.io/techart-2/

## 使い方

- 公開ページを開く、またはリポジトリの `index.html` をブラウザで開くだけで利用できます。
- アセット内蔵のため、オフライン環境でもそのまま表示・操作できます。

## 構成

- `index.html` — アセット内蔵のバンドル済み単一ファイル
